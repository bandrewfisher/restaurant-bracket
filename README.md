# restaurant_app

## Project setup

You will need to create a secrets.json file in the root folder containing your Yelp and Google Maps API keys.

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
